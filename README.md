# Spring 3 MVC (xml) Skeleton

Maven application: Spring 3 MVC using XML-config.

## Components

* Spring MVC v3.2.0
* Thymeleaf v2.0.14
* Bootstrap v2.3.1
* jQuery v1.9.0

## Features

- Internationalization.
- Remember me.
- Simple CRUD operations over two tables.
